# 大衆演劇コミュニティ

## サイト概要
### サイトテーマ
大衆演劇のコミュニティサイト

### テーマを選んだ理由
大衆演劇のコミュニティサイトが無かったので
あれば便利かつもっと多くの情報に触れられると思いました

### ターゲットユーザ
大衆演劇を見に行く人
大衆演劇に興味ある人

### 主な利用シーン
見に行った劇団の感想を発信したい時
見に行く予定の劇団の情報を知りたい時
どんな劇団があるのか知りたい時
